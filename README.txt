☻在list.txt內輸入你想要的歌名
    ☻每一行都會被當作youtube搜尋字串來搜尋，可用空白隔開關鍵字。ex: 周杰倫 你打我媽 我揍你爸
    ☻若覺得以歌名搜尋結果不明確，亦可放入該youtube的連結。
    ☻以驚嘆號開頭，可下載影片(最高畫質) ex:!https://www.youtube.com/watch?v=lWm138oc4X8
    ☻這個文字檔需要以utf-8格式來儲存 (此檔案預設設定即是utf-8)
☻雙擊youtube.exe即開始下載，若程式有閃退的現象，請聯絡作者。
    ☻若給予次程式參數 youtube.exe -get989 [想要的歌單天數，一天有200多首]，可自動爬取好事聯播網989的歌單
☻一個歌名會自動下載前二個搜尋結果，此設定可藉由修改config.ini變更。
☻程式結束後，下載好的歌曲即存放在Songs資料夾內

☻若發現下載不成功，請至https://yt-dl.org/latest/youtube-dl.exe下載更新程式，並放置ffmpeg-3.4.1-win32-static資料夾內。
